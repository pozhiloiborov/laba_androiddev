# Система мониторинга микроклимата умного дома

Встраиваемая система (Raspberry Pi) с подключенными датчиками, выступающая в роли шлюза для сбора данных о микроклимате (температура, влажность)

Android-приложение, которое предоставляет пользователю интуитивный интерфейс для мониторинга показаний в реальном времени, и просмотра показаний в предыдущие моменты времени

Участники:
